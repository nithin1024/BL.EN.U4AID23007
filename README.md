# Backend Evaluation

This repository contains the backend implementation for the evaluation tasks, including:
- Logging Middleware
- Priority Inbox for Notification System
- Vehicle Maintenance Scheduler
- Notification System Architecture Design

## Structure
- `logging_middleware`: Custom logging middleware.
- `notification_app_be`: Python script generating the top 10 priority inbox based on mock data fallback. Includes screenshot.
- `vehicle_maintence_scheduler`: Python script solving the 0/1 Knapsack problem for scheduling maintenance tasks to maximize impact. Includes screenshot.
- `notification_system_design.md`: Markdown document detailing the 6 stages of the Notification System Architecture design.
